# dryrun-todo-app

Build a minimal todo list web application with these features: add todo items with title, mark todos as complete, delete completed todos, filter by status. Use Next.js and TypeScript with PostgreSQL. Deploy to staging only.

## Tech Stack

- **Languages:** TypeScript, JavaScript
- **Frameworks:** Next.js, React, Prisma, Tailwind CSS
- **Database:** PostgreSQL
- **Testing:** vitest
- **Linting:** eslint
- **Package Manager:** npm
- **Features:** בניית יכולת פיתוח מינימלית, הטמעת תהליך staging ו-deployment, פיתוח מוקדם של features ליבה

## Getting Started

```bash
# Clone the repository
git clone https://github.com/KBRglobal/dryrun-todo-app.git
cd dryrun-todo-app

# Install dependencies
npm install

# Start development server
npm run dev
```

## Docker

```bash
docker compose up
```
